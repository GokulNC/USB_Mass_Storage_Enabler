A quoi sert cette application :


Cette application pour appareil ROOTÉ sert à monter votre carte mémoire en tant que périphérique USB de Stockage de Masse (USB Mass Storage - UMS) sur votre ordinateur, votre TV, votre lecteur Audio/DVD ou n'importe quoi d'autre.
Cette application NE montera PAS votre stokage interne (puisqu'il est contenu dans le répertoire ./media/0/ de la partition /data et que la partition /data est formatée en EXT4).
Si vous souhaitez vraiment monter votre stockage interne en tant que Stockage de Masse, cela peut être réalisé en utilisant l'application USB Sharer (mais il sera monté en lecture seule).

Comment utiliser l'application :


1. Pour activer le Stockage de Masse, cliquer sur 'Activer Stockage de Masse'. (Ré-essayer 2-3 fois si ça ne fonctionne pas, éssayer également de modifier les paramètres)

2. Avant de débrancher le câble USB, éjecter le périphérique depuis l'ordinateur en premier, puis cliquer sur le bouton 'Déconnecter Stockage de Masse'.

Raisons probables du disfonctionnement de cette application :


1. Le noyau (kernel) de votre appareil n'embarque pas le Gadget «Mass Storage Gadget» avec le support de l'émulation USB.
Solution : Demander à un développeur de votre appareil de compiler un noyau embarquant le Gadget «Mass Storage Gadget».

2. Votre appareil ne supporte pas les cartes mémoires, ou aucune carte mémoire n'est insérée.

3. Votre carte mémoire est peut être cryptée ou utilisée en tant que Stockage Interne (à partir d'Android 6.0, le stockage interne est formatté en tant que partition EXT4 et crypté)
Solution : Vérfier que la carte mémoire soit formatée en tant que partition FAT32.

4. Il peut y avoir quelque chose (comme un Anti-Virus) qui n'autorise pas à désactiver la protection SE Linux.

5. Votre device pourrait ne pas être rooté correctement.
